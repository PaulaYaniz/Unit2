## Create a function that produces the table of Truth for 3 inputs:
![image](https://user-images.githubusercontent.com/89135778/202064073-ab4a9f0a-24f1-48cd-adc3-d5590b1bf430.png)

```.py
def get_truth():
    a = True
    b = True
    c = True
    out = "A | B | C\n"
    
    for i in range(8):
        if i % 4 == 0:
            a = not a
        if i % 2 == 0:
            b = not b
        c = not c
        out += f"{int(a)} | {int(b)} | {int(c)}\n"
        
    return out


table = get_truth()
print(table)
```
